fn register_callback<F>(&mut self, callback: F) -> Self::CallbackHandle
where F: FnMut(),

Creates a new callback that would be called when stop would be requested.

Callback would be called on same thread which did request stop. If token would already have stop_requested() == true at time of creation of callback, then callback would be called on thread creating stop callback.

Dyn Compatibility§
